# 직장 내 괴롭힘 사측 조사 체크리스트

근로기준법 제76조의2, 제76조의3 기준으로 사측 조사 절차를 점검할 수 있도록 만든 정적 웹페이지입니다.

배포 주소:

`https://ssonggl2.github.io/coding/`

## 포함 내용

- 신고 접수 및 사건 등록
- 조사기간 중 피해자 보호조치
- 조사기구 구성 및 비밀유지 고지
- 사실조사 수행
- 법률요건 판단
- 조사결과에 따른 조치
- 불리한 처우 금지 및 사후관리
- 문서화 및 보관
- 법적 리스크 점검

## 파일 구성

- `index.html`: GitHub Pages로 배포되는 메인 페이지
- `직장내괴롭힘_조사체크리스트.md`: 초기 기획 및 문안 정리 파일

## 로컬에서 확인

브라우저에서 `index.html`을 직접 열면 바로 확인할 수 있습니다.

## 배포 방식

이 저장소는 GitHub Pages에서 `main` 브랜치의 `/(root)` 기준으로 배포합니다.

## 주의

- 실제 사건기록, 개인정보, 민감한 조사자료는 저장소에 커밋하지 않는 것을 권장합니다.
- 이 페이지는 내부 조사 실무용 점검도구이며, 최종 법률판단은 노무·법률 전문가 검토를 병행해야 합니다.
